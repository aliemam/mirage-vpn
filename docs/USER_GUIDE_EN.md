# Mirage VPN User Guide

## Installing the App

1. Download the APK file
2. If received via Bluetooth or SD card, tap the file
3. If prompted about "unknown sources", allow installation
4. Install the app

## Usage

### Connecting
1. Open Mirage VPN app
2. Tap the big **Connect** button
3. If first time, approve VPN permission when asked
4. Wait for the indicator to turn green (Connected)

### Disconnecting
- Tap the **Disconnect** button

## FAQ

### Why is the connection slow?
This tunnel uses DNS which has speed limitations. It's sufficient for normal browsing and messaging.

### Is it secure?
Yes. All traffic is encrypted and routed through DNS tunneling.

### Why does it need VPN permission?
This is required to route your traffic through the secure tunnel.

## Troubleshooting

If you have issues:
1. Close and reopen the app
2. Check your regular internet connection
3. Restart your phone
